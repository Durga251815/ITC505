Addendum

  • The process of creating the "Revenge of Lights Out" game provided valuable insights into blending creativity with problem-solving.
  • The primary objective was to craft an interactive and captivating game that presents an enjoyable challenge to players.

Creative Decisions:

  • Game Design: Opting for a 5x5 grid aimed to offer a balanced level of difficulty, catering to players with different skill levels while maintaining a manageable complexity.
  • User Interface: Embracing a minimalist design approach for the interface focused on enhancing gameplay immersion. This choice aligned with contemporary design trends emphasizing simplicity and user-friendliness.
  • Feedback System: Implementing a modal window to signify game completion aimed to deliver a clear and rewarding experience for players.

Challenges and Solutions:

  • Board Randomization: Initially, generating randomized board layouts posed a challenge.
    To tackle this, a randomization function was introduced to ensure unique game setups in each session.
  • Playability Assurance: Ensuring that the game remained solvable post-randomization required meticulous testing and refining of the shuffling algorithm.

Key Learnings:

  • User-Centric Design: Emphasizing user experience proved pivotal. Simple and intuitive gameplay and interface design significantly heightened player engagement.
  • Testing Importance: Thorough testing played a crucial role in identifying and resolving gameplay and usability issues, underscoring the significance of comprehensive testing in game development.
