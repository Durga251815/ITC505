Addendum

Project Overview: The web page includes a currency conversion tool for converting money between USD and INR. Users can input an amount, choose the conversion direction, and get the converted amount in the respective currency as output.

Implementation Details: The JavaScript-based currency converter is designed to gather user input from an HTML form, perform the conversion process in JavaScript according to the chosen option (such as from INR to USD or vice versa), and then showcase the outcome on the webpage.

Challenges and Solutions: Ensuring precise currency conversion and managing invalid inputs posed a significant challenge during development. To address this, input validation was implemented alongside accurate currency conversion formulas. Another hurdle was crafting a responsive and user-friendly interface, which was successfully achieved through the use of CSS Grid layout and contemporary styling methods.

Learning Outcomes: This project enhanced my knowledge of JavaScript event handling, processing form inputs, and creating responsive web designs using CSS Grid. It also underscored the significance of user input validation and providing clear feedback to users in web applications.
